# Flappy Bird (Godot Engine)

A simple **Flappy Bird clone** made in the [Godot Engine](https://godotengine.org/).  

---

## Gameplay

Press **Space** to make the bird flap and avoid the pipes.

---

## Features

- Smooth physics-based flapping
- Randomly generated pipes  
- Score tracking  
- Restart on game over  

---

## Built With

- **Engine:** [Godot 4.x](https://godotengine.org/download)
- **Language:** GDScript

---
